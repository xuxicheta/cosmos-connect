Рабочая группа

по разработке систем управления комплексами и специализированного программного обеспечения

Структура рабочей группы:

  • группа системного проектирования;
  • группа прикладного программирования;
  • группа разработки алгоритмического обеспечения.

Направления деятельности рабочей группы:

  1. Комплексная разработка изделий, начиная от разработки технических заданий (ТЗ), структурных схем, протоколов, технических условий (ТУ) и заканчивая проведением испытаний и сдачей Заказчику.
  2. Разработка радиолиний, включая расчётов параметров и моделирование сквозного тракта.
  3. Разработка систем управления радиолокационных и связных комплексов, включая встроенное и прикладное программное обеспечение (ПО).
  4. Разработка контрольно-проверочной аппаратуры (КПА) и технологического ПО.
  5. Разработка алгоритмов синтеза и обработки радиолокационных изображений и их программная реализация для перспективных вычислительных архитектур.
  6. Разработка встроенного ПО для конкретных блоков или устройств, в том числе систем реального времени.
  7. Разработка ПО баз данных (БД) для клиент-серверных архитектур.
  8. Прикладное программирование на языках высокого уровня. Разработка ПО под операционные системы Windows/Linux (в том числе – МСВС, Astra Linux Special Edition и т.п.).
  9. Разработка Web приложений.
  10. Разработка конструкторской и программный документации в соответствии с требованиями ЕСКД и ЕСПД.

Краткий перечень «последних» работ, выполненных рабочей группой:

  1. Спутниковые системы связи
    Разработка встроенного ПО систем управления изделиями.
    • Разработка алгоритмического обеспечения изделий.
    • Разработка модуля КПА обеспечивающего подсчет достоверности в каналах связи в реальном масштабе времени.
    • Проведение испытаний изделий
    • Разработка встроенного и клиентского ПО центров управления связью.
    • Пуско-наладочные работы.
    • Служба поддержки.
  2. Комплекс вычислительной аппаратуры
    • Отработка и макетирование аппаратуры изделия
    • Разработка специального программного обеспечения (СПО) изделий (АРМ, экран коллективного пользования, центральный вычислитель).
    • Разработка программного сервиса диагностики аппаратного обеспечения
    • Разработка программного сервиса вторичной радиолокации
  3. Системы радиолокации
    • Разработка ПО алгоритмов синтезирования РЛИ на графических ускорителях
    • Разработка алгоритмов синтеза и обработки РЛИ.
    • Разработка прикладного и специализированного ПО.
    • Разработка программы проведения испытаний.
  4. Система тахографического контроля за соблюдением режима труда и отдыха водителей в транспортном комплексе РФ, функционирующей на базе использования системы ГЛОНАСС
    • Разработка и согласование протоколов взаимодействия.
    • Разработка БД.
    • Разработка и программирование имитаторов.
    • Разработка программы и методики испытаний.
    • Разработка ПО подсистем – сбора сведений, анализа информации, хранения данных.
  5. Модернизация мобильных станций спецсвязи
    • Модернизация ПО управления и GPS позиционирования станций.
    • Разработка специализированного центрального узла картографической привязки абонентов.
    • Разработка программной документации и участие в проведении испытаний.
  6. Разработка прикладного пользовательского ПО для источников бесперебойного питания
    • Разработка архитектуры СПО и протоколов взаимодействия с ИБП по интерфейсам - RS-232, USB, Ethernet.
    • Разработка ПО оператора, ПО администратора сети, ПО автоматического отключения абонентов, БД хранения управляющей и телеметрической информации.
    • Разработка документации по ЕСПД, включая руководство оператора и программу и методику испытаний.
